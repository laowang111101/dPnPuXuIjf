# 员工管理系统 java1205

## 项目概述

员工管理系统是基于Spring Boot和Vue的前后端分离系统。该系统支持管理员和员工两种角色，覆盖了从个人中心到薪资管理的多种功能模块。

## 技术栈

### 后端技术

- Spring Boot
- MyBatis

### 前端技术

- Vue
- ElementUI
- Axios
- ECharts

### 运行环境

- Java JDK 1.8
- Node 12
- MySQL

## 功能模块

### 系统角色

- 管理员
- 员工

### 功能介绍

- **个人中心**
  - 修改密码
  - 个人信息更新修改

- **员工管理**
  - 员工数据信息查询、新增、修改、删除

- **部门管理**
  - 部门数据信息查询、新增、修改、删除

- **职位管理**
  - 职位数据信息查询、新增、修改、删除

- **员工考勤类型**
  - 员工考勤类型信息查询、新增、修改、删除

- **请假类型**
  - 请假类型数据信息查询、新增、修改、删除

- **薪资管理**
  - 薪资数据增删改查

- **员工考勤管理**
  - 员工考勤数据信息查询、新增、修改、删除、报表分析

- **员工请假管理**
  - 员工请假数据信息查询、新增、修改、删除、报表分析

- **公告管理**
  - 公告数据增删改查

## 部署步骤

1. 确保Java JDK 1.8、Node 12和MySQL环境已安装并正确配置。
2. 导入MySQL数据库脚本，初始化数据。
3. 使用IDEA等开发工具导入项目代码，运行后端服务。
4. 前端项目通过npm安装依赖，运行构建，启动前端服务。

## 目录结构

```plaintext
员工管理系统
├── backend                # 后端代码
├── frontend               # 前端代码
│   ├── node_modules       # 前端依赖模块
│   ├── public             # 静态资源文件
│   └── src                # 源代码
└── sql                    # 数据库脚本
```
